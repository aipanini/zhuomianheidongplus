# Desktop Blackhole Plus

在你的电脑桌面上放一个真正的黑洞！它会跟着鼠标移动，把桌面上的文件"吞噬"进去，还能让周围的画面像被引力弯曲一样扭曲变形。

## 下载

**exe 下载地址**：https://github.com/aipanini/desktop-blackhole-plus/releases/download/v1.0/DesktopBlackholePlus.exe

下载后双击运行即可，无需安装 Python 或任何其他软件。

> 也可以在 [Releases 页面](https://github.com/aipanini/desktop-blackhole-plus/releases) 查看。

## 这是什么？

一个桌面趣味小程序。运行后：

- 桌面上会出现一个黑洞，缓慢跟随你的鼠标移动
- 黑洞中心的区域是纯黑的（连光都逃不出来）
- 黑洞周围有一圈发光的旋转光环
- 靠近桌面上文件时，文件会被"吸入"黑洞（实际是被移动到了一个文件夹里，可以恢复）
- 黑洞周围的画面会发生扭曲，就像真正的引力透镜效果
- 每吞噬一个文件，黑洞都会稍微变大一点
- 当黑洞达到最大体积时，会触发彩蛋：屏幕震动 + 背景音乐

虽然黑洞覆盖在桌面上方，但你的鼠标点击会直接穿透它，正常操作桌面，就像它不存在一样。

## 怎么用？

### 控制方法

| 操作 | 作用 |
|------|------|
| **F9** | 显示 / 隐藏黑洞 |
| **F2** | 打开 / 关闭控制台（浏览器页面） |
| **Ctrl+Alt+↑** | 增大黑洞 |
| **Ctrl+Alt+↓** | 缩小黑洞 |
| **Ctrl+Alt+Q** | 退出程序 |

也可以右键点击任务栏右下角的黑洞图标，选择：
- 显示 / 关闭黑洞
- 显示 / 关闭控制台
- 退出

> **找不到黑洞图标？** Windows 11 会把托盘图标收起来，点击任务栏右下角的 **^** 箭头展开就能看到。

> **控制台页面关了怎么办？** 按 F2 就会重新用浏览器打开控制台，所有设置都还在。

### 控制台说明

控制台是一个网页，会用系统默认浏览器自动打开。里面可以调节：

- **黑洞外观**：黑洞大小、扭曲范围、扭曲强度、光环亮度
- **视觉风格**：金色、炽焰、经典橙红、幽冥青、紫电 五种风格
- **鼠标跟随**：跟随速度和滑动惯性
- **吞噬功能**：开关吞噬、吞噬范围、吞噬速度、文件存放路径
- **一键预设**：微型、标准、巨型、狂暴 四种快捷配置

每个调节项下面都有通俗易懂的说明文字。拖动滑块会实时生效，不需要点保存。

### 吞噬的文件去哪了？

被黑洞吞噬的文件会被移动到 exe 所在目录下的 `资源/被吞噬/` 文件夹里。你可以：

1. 在控制台点"恢复所有文件"按钮
2. 或者手动去那个文件夹里找

### 彩蛋

当黑洞吞噬足够多的文件、达到最大体积时，会触发彩蛋：
- 屏幕剧烈震动 3 秒
- 播放背景音乐
- 缩小黑洞后可以再次触发

## 系统要求

- Windows 10 (2004 版本以上) 或 Windows 11
- 系统需要有浏览器（Windows 自带的 Edge 就可以）

## 常见问题

**屏幕全黑了？**
少数显卡可能有兼容性问题。关掉程序，重新打开试试。

**黑洞周围有拖影/残影？**
说明系统版本较旧。升级到 Windows 10 2004 以上版本即可解决。

**热键没反应？**
可能被其他软件占用了。试试用托盘菜单控制。

**吞噬不生效？**
确保桌面上有文件（不是快捷方式），并且黑洞大小够大。黑洞需要靠近文件才能吞噬。

**程序卡顿？**
在控制台里调小黑洞大小，或者选择"微型"预设。

**控制台打不开？**
控制台运行在本地 `http://127.0.0.1:18089`。如果浏览器没有自动打开，手动在浏览器地址栏输入这个地址即可。也可能是杀毒软件拦截了本地端口，添加信任即可。

## 开发者信息

如果你是开发者，想自己运行或修改代码：

```bash
pip install -r requirements.txt
python main.py
```

项目结构：
```
├── main.py                        # 主程序（OpenGL 渲染 + 主循环）
├── config.py                      # 集中参数配置
├── winutil.py                     # Windows 系统工具（DPI、热键、窗口样式）
├── tray.py                        # 系统托盘
├── swallow.py                     # 文件吞噬模块
├── web_panel.py                   # Web 控制台（HTTP 服务器 + 前端页面）
├── requirements.txt               # 依赖列表
├── desktop-blackhole-plus.spec    # PyInstaller 打包配置
├── shaders/                       # OpenGL 着色器
│   ├── lens.vert
│   └── lens.frag
└── 资源/                          # 资源文件
    ├── bgm_easter.wav             # 彩蛋背景音乐
    └── 被吞噬/                    # 被吞噬文件的存放目录（自动创建）
```

打包 exe：`pyinstaller desktop-blackhole-plus.spec`

### 技术说明

控制台采用 Web 技术实现（Python 标准库 `http.server` + 内嵌 HTML/CSS/JS），无需额外依赖。相比传统 GUI 框架，浏览器原生支持 DPI 缩放和响应式布局，在任何分辨率的屏幕上都能清晰显示。
